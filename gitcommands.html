Commands used to connect git to github and push files

THe first things to be done are first to download git and install it, another crusial thing to do is to create a github account.

Once these are done you can open your terminal.

The first thing to do is to set your terminal unde your name and email.Use git config:

    $ git config --global user.name "Santa Claus" $ git config --global user.email santa@northpole.com

Next thing to do is to create a directory for your file then create the file

Once the file is created. you can go to step of initiating git and then continue up to when you push you file to github." >

    Initiate git. $ git init The next step is to add you file to git.if it's all the files in the directory $ git add ., if it's only one file $ git add filename Next you have to commit changes. $ git commit -m "add a short, descriptive present-tense message here describing the changes made" After commiting the changes the next thing is to connect git to your github repository. $ git remote add [remote name] [remote URL]. ex: $ git remote add origin https://github.com/moringaschool/hello-world.git The last thing to do is to push your changes in github. $ git push [remote name] [branch name]. ex:$ git push origin master
